description: NEVER submit predictions or code to competition server without explicit user approval
type: feedback
---

NEVER submit anything to the competition server (ainm.no API, ZIP uploads, or any external submission endpoint) without explicitly asking the user first and getting confirmation.

**Why:** Competition submissions are limited (3/day for image task). Accidental or premature submissions waste precious attempts and could submit bad results to the leaderboard.

**How to apply:** Before ANY action that sends data to an external server (competition API, leaderboard, task submission), STOP and ask the user for permission. This applies to all three repos and all competition tasks.
